**Epic:** 3 — Dashboard MVP
**Agente:** @dev
**Status:** Done

## User Story

Como Ryan, quero gerenciar os horários de trabalho, serviços oferecidos, bloqueios de data e configurações gerais pelo dashboard, para que o agente WhatsApp mostre informações sempre atualizadas.

## Acceptance Criteria

- [x] 4 tabs: Geral, Horários, Serviços, Bloqueios
- [x] Geral: formulário com todas as configurações (nome, telefone, intervalo, antecedência, lembrete, mensagem boas-vindas)
- [x] Horários: listar todos os dias da semana com hora_inicio/hora_fim, toggle ativo/fechado, edição inline
- [x] Serviços: listar com preço e duração, toggle ativo/inativo, adicionar novo
- [x] Bloqueios: listar períodos bloqueados com motivo, adicionar e deletar
- [x] Mudanças em horários/bloqueios refletem imediatamente nos slots disponíveis
