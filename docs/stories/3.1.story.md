# Story 3.1 — Hoje — Timeline + Stats + Realtime

**Epic:** 3 — Dashboard MVP
**Agente:** @dev
**Status:** Done
**Branch:** main

## User Story

Como Ryan, quero ver em tempo real os agendamentos do dia com uma timeline visual e estatísticas rápidas (total de agendamentos, receita prevista, concluídos), para que eu saiba exatamente o que acontece na barbearia hoje.

## Acceptance Criteria

- [x] Página /hoje mostra agendamentos do dia (excluindo cancelados)
- [x] StatCards: total de agendamentos, receita prevista, concluídos
- [x] Timeline com cada agendamento: hora, cliente, serviço, valor, status
- [x] Badge de status colorido (amber=pendente, green=confirmado, blue=concluido, red=cancelado, purple=faltou)
- [x] Supabase Realtime: novos agendamentos via WhatsApp aparecem sem reload
- [x] Estado vazio com mensagem quando não há agendamentos
- [x] Data do dia formatada em ptBR na página
- [x] Loading skeleton
