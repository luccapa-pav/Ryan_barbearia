# Story 3.2 — Agendamentos — List + CRUD completo

**Epic:** 3 — Dashboard MVP
**Agente:** @dev
**Status:** Done
**Branch:** main

## User Story

Como Ryan, quero visualizar todos os agendamentos em uma tabela com filtros, criar novos agendamentos pelo dashboard com seleção de cliente/serviço/slot, editar e cancelar agendamentos existentes.

## Acceptance Criteria

- [x] Tabela paginada de agendamentos (20 por página) com ordenação por data desc
- [x] Filtros: por status, por data
- [x] Limpar filtros
- [x] Paginação com indicador de página
- [x] Botões de ação: editar, concluir, cancelar (quando aplicável)
- [x] Sheet "Novo agendamento": busca de cliente, seleção de serviço, data, slots disponíveis, observações
- [x] Sheet "Editar": pré-preenchido com dados existentes
- [x] Slots calculados via buscarSlotsDisponiveis (reutiliza lib/slots.ts)
- [x] Mensagem de estado vazio na tabela
- [x] Toasts de confirmação/erro
