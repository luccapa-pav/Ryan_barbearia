# Story 2.3 — lembrete-flow (dia + hora)

**Epic:** 2 — WhatsApp Agent
**Agente:** @dev
**Status:** Todo
**Branch:** feature/whatsapp-agent

## User Story

Como Ryan, quero que o sistema envie lembretes automáticos para os clientes no dia anterior e 1 hora antes do agendamento, para que o número de faltas diminua.

## Acceptance Criteria

- [ ] Cron 1: executa às 20:00 diariamente, envia lembretes para agendamentos do dia seguinte
- [ ] Cron 2: executa a cada 30min, envia lembretes para agendamentos em 55-65min
- [ ] Apenas agendamentos com status pendente|confirmado recebem lembretes
- [ ] Cron 1 marca lembrete_dia=true após envio (evita duplicação)
- [ ] Cron 2 marca lembrete_hora=true após envio (evita duplicação)
- [ ] Mensagem do cron 1: "Olá {nome}! Lembrete: amanhã às {hora} você tem {serviço} com Ryan."
- [ ] Mensagem do cron 2: "Olá {nome}! Seu agendamento de {serviço} é em 1 hora ({hora}). Te esperamos!"
