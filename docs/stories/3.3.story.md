# Story 3.3 — Clientes — List + histórico

**Epic:** 3 — Dashboard MVP
**Agente:** @dev
**Status:** Done
**Branch:** main

## User Story

Como Ryan, quero ver todos os meus clientes em uma lista pesquisável e, ao clicar em um, ver seu histórico de agendamentos, total gasto e observações, para que eu possa conhecer melhor minha clientela.

## Acceptance Criteria

- [x] Lista paginada de clientes (20 por página) ordenada por nome
- [x] Busca em tempo real por nome ou telefone
- [x] Avatar com inicial do nome
- [x] Painel lateral com histórico ao selecionar cliente
- [x] Histórico: últimos 10 agendamentos com status, serviço e data
- [x] Total gasto (soma de agendamentos concluídos)
- [x] Número de visitas concluídas
- [x] Observações do cliente exibidas se existentes
- [x] Estado vazio quando nenhum cliente encontrado
