# Story 2.2 — agendamento-flow + session state

**Epic:** 2 — WhatsApp Agent
**Agente:** @dev
**Status:** Todo
**Branch:** feature/whatsapp-agent

## User Story

Como cliente, quero agendar um horário na barbearia pelo WhatsApp em uma conversa guiada, para que eu possa marcar sem precisar ligar ou ir pessoalmente.

## Acceptance Criteria

- [ ] Fluxo multi-turn completo: nome → serviço → data → horário → confirmação
- [ ] Estado da conversa salvo em sessoes_whatsapp (TTL 30min)
- [ ] Se cliente já existe, vai direto para escolha de serviço
- [ ] Lista numerada de serviços disponíveis (ativo=true)
- [ ] Lista de datas com slots disponíveis (próximos 7 dias configuráveis)
- [ ] Slots calculados corretamente (horários - bloqueios - agendamentos existentes)
- [ ] Confirmação com resumo: serviço, data, hora, preço
- [ ] INSERT em agendamentos com origem='whatsapp', status='confirmado'
- [ ] Mensagem de confirmação enviada via WhatsApp
- [ ] Sessão limpa após confirmação ou cancelamento

## Estados da Sessão

```
INICIO → AGUARDANDO_NOME → ESCOLHER_SERVICO → AGUARDANDO_SERVICO
→ ESCOLHER_DATA → AGUARDANDO_DATA → ESCOLHER_HORA → AGUARDANDO_HORA
→ AGUARDANDO_CONFIRMACAO → [FIM]
```
